# 🚀 TrendPulse Project

## 📌 Overview
**TrendPulse** is a Python-based data pipeline project designed to **collect, process, analyse, and visualise trending stories data**.  
The project is structured as a sequence of tasks that transform raw trending data into meaningful insights and visual dashboards.

This repository contains:
- Data collection scripts  
- Data cleaning and analysis logic  
- Visualisations generated from analysed trends

---

## 🧩 Task Breakdown

### ✅ Task 1 — Data Collection
- Collects trending stories from the source
- Stores raw data in CSV format

### ✅ Task 2 — Data Processing
- Cleans and structures the dataset
- Handles missing or inconsistent values

### ✅ Task 3 — Data Analysis
- Performs statistical and categorical analysis
- Produces the processed dataset:  
  `data/trends_analysed.csv`

### ✅ Task 4 — Data Visualisation
Creates multiple insights using **Matplotlib** and **Seaborn**:
- Top 10 stories by score
- Number of stories per category
- Score vs comment count scatter plot
- Combined dashboard image

All outputs are saved in the `outputs/` folder.
